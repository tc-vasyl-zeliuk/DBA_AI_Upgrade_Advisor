# Capability Map

## Core capability areas

### Inventory and normalization
- discover managed data assets
- normalize asset identity and metadata
- retain account, region, environment, and ownership context

### Lifecycle knowledge
- store support timelines
- map versions to support phase
- retain vendor source references and refresh metadata

### Policy and scoring
- classify compliance and urgency
- model cost exposure
- compute severity and remediation priority

### Upgrade assessment
- identify likely blockers and checks
- recommend target versions when evidence exists
- attach confidence and rationale

### AI brief generation
- generate concise per-asset briefs
- draft leadership-facing portfolio summaries
- summarize evidence without becoming the source of truth

### Workflow outputs
- emit backlog candidates
- generate ticket or change record drafts
- publish alerts and summaries to collaboration channels
